API Documentation

Built for quick integration

Use one endpoint to fetch every name, a single record, a random result, or filtered matches by keyword.

Base Endpoint

/api/asmaul-husna

Returns all 99 names by default in a consistent JSON shape.

Query Parameters

  • lang - `english`, `bangla`, `urdu`, `indonesian`
  • id - fetch a specific name by number, for example `1`
  • search - search Arabic text, transliteration, or English meaning
  • random - set to `true` to get one random name

Example Requests

GET /api/asmaul-husna?id=1
GET /api/asmaul-husna?search=mercy
GET /api/asmaul-husna?random=true&lang=urdu

Response Shape

{
  "number": 1,
  "name": {
    "arabic": "...",
    "transliteration": "...",
    "translated": "..."
  },
  "meaning": "...",
  "audio_url": "...",
  "reference_url": "..."
}
